System analizy ryzyka kredytowego (Credit Risk Analysis System):
1. Skrypt w Pythonie do oceny profili ryzyka
- Modele scoringowe
- System klasyfikacji dłużników
2. Narzędzia do modelowania spłat (Repayment Modeling Tools):
- Predykcyjne modele spłat w R lub Pythonie
- Analiza historycznych wzorców spłat
- Kalkulatory NPV dla portfeli wierzytelności
3. System wyceny wierzytelności (Debt Valuation System):
- Automatyczny kalkulator cen zakupu długów
- Model ryzyka-zwrotu
- Narzędzia do symulacji różnych scenariuszy
4. Dashboardy analityczne (Analytical Dashboards):
- Interaktywne wizualizacje portfela
- Raporty efektywności windykacji
- Monitoring KPI
5. System zarządzania rezerwami (Provisions Management System):
- Model kalkulacji rezerw
- Narzędzia do stress-testów
- System monitoringu adekwatności rezerw
6. Baza danych SQL (SQL Database):
- Schema dla przechowywania danych o dłużnikach
- System śledzenia historii spłat
- Struktura do analizy portfela
